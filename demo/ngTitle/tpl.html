Use pageTitleProvider.setPostfix('Demo') configuration to set the global website title.

Use pageTitleProvider.setDelimiter(' | ') configuration to set the title components delimiter, default is ' - '.

Use {title: 'Route Title'} in the route object to set the title component for the route.

Use 'pageTitle.setTitle('Controller Title')' service to set the title component from the controller.

Use 'data-page-title' attribute to set the title component from the template.

All options are compatible and interchangable.
